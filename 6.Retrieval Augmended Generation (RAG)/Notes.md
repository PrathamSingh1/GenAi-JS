Retrieval-Augmented Generation(RAG) -: A Technique where a model first retrieves relevant information from a knowledge source, then uses that information to generate an answer.

Similarity Search -

Similarity search is when you find things that are most similar to a given input based on meaning, not just exact words.

Vector embeddings -

Vector embeddings are numbers that represent the meaning of text/data so that similar meanings have similar numbers.

To store this type of embeddings we use the vector database and to search in this type of databases we use the vecter-index data structure and use the cosine_similarity algorithm.
